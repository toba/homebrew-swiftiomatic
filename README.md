# Homebrew Tap for swiftiomatic

This is the official Homebrew tap for [swiftiomatic](https://github.com/toba/swiftiomatic), an AST-based Swift code analysis CLI.

## Installation

```bash
brew tap toba/swiftiomatic
brew install swiftiomatic
```

## Updating

```bash
brew update
brew upgrade swiftiomatic
```

## Uninstalling

```bash
brew uninstall swiftiomatic
brew untap toba/swiftiomatic
```

## Requirements

- macOS Sequoia (15.0) or later
- Xcode 16.0 or later
